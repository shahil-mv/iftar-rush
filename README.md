# Iftar Rush: Office Edition - Run Guide

To run this project, follow these steps:

### 1. Prerequisites
Ensure you have **Python 3** installed on your system.

### 2. Install Dependencies
This game requires the `pygame` library. Install it using pip:
```bash
pip install pygame
```

### 3. Run the Game
Execute the main script from the project root:
```bash
python3 main.py
```

### Controls
- **Movement**: Use **Arrow Keys** (Left/Right) or **A/D Keys**.
- **Alternative Movement**: Move the **Mouse** to position the player.
- **Start Game**: Press **SPACE** on the start screen.
- **Restart**: Press **R** on the game over screen.
